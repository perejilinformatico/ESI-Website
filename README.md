#  ESI Web — Educación Sexual Integral

![ESI Banner](https://via.placeholder.com/1200x300?text=ESI+Web)

##  Descripción

**ESI Web** es una plataforma educativa desarrollada para informar, concientizar y facilitar el aprendizaje sobre la **Educación Sexual Integral (ESI)** de manera accesible, visual e interactiva.

El objetivo principal del proyecto es ofrecer contenido claro y atractivo para adolescentes y jóvenes, combinando diseño moderno, animaciones y una experiencia de usuario dinámica.

## Características principales

* **Diseño moderno y atractivo** con Tailwind CSS
*  **Animaciones fluidas** utilizando GSAP
*  **Responsive Design** adaptable a dispositivos móviles
*  **Contenido educativo claro y dinámico**
*  **Interfaz interactiva** pensada para jóvenes
*  **Alto rendimiento** gracias a Astro

##  Instalación y uso

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/esi-web.git

# Entrar al proyecto
cd esi-web

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
```
